# Agent Skills

This folder contains examples of [Agent Skills](https://agentskills.io/), an open format for packaging instructions, scripts, examples, and resources that AI agents can load on demand.

## Examples

- `project-notetaker/`: A custom skill for formatting unstructured meeting notes into a structured project report.

## Usage

Agent skills are designed to be discovered and loaded by compatible AI agents (such as Claude Desktop, Gemini CLI, or other agentic systems). To use a skill, point your agent to this directory or copy the skill folder to the agent's configured skills path.
