# Project Notetaker

## Instructions
When the user provides meeting notes, follow this structure:
1. Title: A clear name for the meeting.
2. Attendees: List all participants.
3. Key decisions: A bulleted list of what was decided.
4. Action items: A table with columns for task, owner, and due date.
