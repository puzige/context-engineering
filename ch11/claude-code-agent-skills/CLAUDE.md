# Claude Code Instructions

Assume [agent-skills](https://github.com/addyosmani/agent-skills) plugin is already installed.

- Use `/spec` to frame the change.
- Use `/plan` to break the work into small steps.
- Use `/build` to implement the minimal code.
- Use `/test` to run the Python unit tests.
- Use `/review` to check the result before finishing.
- Keep the code minimal.
- Preserve the weighted `idea-score` rule described in `agent-skills-workflow.md`.
