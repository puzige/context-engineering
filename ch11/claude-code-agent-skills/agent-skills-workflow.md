# Agent Skills Workflow

Public skill pack repo: `https://github.com/addyosmani/agent-skills`

## Install

```text
/plugin marketplace add addyosmani/agent-skills
/plugin install agent-skills@addy-agent-skills
```

## Lifecycle commands

- `/spec`
- `/plan`
- `/build`
- `/test`
- `/review`

## Fixed feature rule

- Formula: `impact * 5 + strategic_fit * 3 - effort * 2`
- Validation: each numeric field must be between `0` and `5` inclusive
