# Zapier

This chapter shows Zapier as hosted trigger-action automation around AI steps.

## Workflow manifests

- `support_routing/zap.json`: support intake flow that classifies a request and sends it to the right team
- `lead_qualification/zap.json`: lead triage flow that scores and routes prospects for follow-up
- `ops_notifications/zap.json`: operations alert flow that turns AI interpretation into the right notification path
- `approval_chain/zap.json`: approval flow that routes work through the right reviewers before action
