# Zapier support routing recipe

This example shows how Zapier can connect an AI decision step to business apps.
It is documented as a recipe because the platform lives primarily in the hosted UI.

## What it demonstrates

- Trigger-action automation around an AI step
- Tool use outside the chat window
- Business process integration with guardrails

## Manifest

Inspect `zap.json` for the step layout.

See `recipe.md` for the step-by-step routing sequence that the manifest mirrors.
